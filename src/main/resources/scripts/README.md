# Short version of the installation instructions

1. Obtain valid license files for IS (Integration Server), UM (Universal Messaging), and TS (Unit test suite). By default,
   these are assumed to be located in /opt/wm/installation as *is-license.xml*, *um-license.xml*, and *ts-license.xml*.

2.  Edit the file **settings.sh**.
   
   Check, and update (if necessary), the values for *WEBMETHODS_VERSION*, *LICENSE_FILE_IS*, *LICENSE_FILE_UM*,
   *LICENSE_FILE_TS*, *EMPOWER_USERID*, and *EMPOWER_PASSWORD*.
